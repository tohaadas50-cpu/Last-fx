# Brand Voice: The "Real" Friend

We do not speak like corporations. We speak like a smart, honest friend.
We never use jargon like "synergy," "unlock," or "elevate."
We prefer short sentences. We love paradoxes.

## Key Principles:
1. Don't sell the solution, sell the problem properly understood.
2. Honor the user's intelligence.
3. If it sounds like an ad, rewrite it.
